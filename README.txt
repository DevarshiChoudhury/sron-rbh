This repository consists of 3 sub repositories:

1. data_filtration: contains the jupyter notebook "Filtration_and_offset.ipynb" and the necessary input files which are used to filter the X-ray and optical data based on the different criteria specified in the report.

2. image_gen: contains the jupyter notebook "Image_generator.ipynb" and the necessary input files which are used to obtain the PanSTARRS images and overplot them with X-ray source position and Petrosian radius.

3. lum_cal: contains the jupyter notebook "luminosity_calculator.ipynb" and the necessary input files which are used to obtain the luminosity values for X-ray targets based on their redshift and flux information.
